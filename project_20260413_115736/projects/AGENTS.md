# 异界寻钥 - 场景解谜游戏

## 项目概述
这是一个基于HTML5的场景切换解谜游戏。玩家需要在两个相似的场景间切换，通过寻找钥匙来打开对应的门，最终到达终点。

## 游戏规则

### 核心机制
- **场景切换**：玩家可以在两个场景间自由切换
- **钥匙收集**：两把钥匙（金钥匙、银钥匙）分布在不同的场景中
- **门解锁**：对应的钥匙可以打开对应的门
- **通关条件**：打开所有门后到达终点

### 地图设计
- **场景1**：银门在中央通道，银钥匙在右上角开放房间
- **场景2**：金门在中央通道，金钥匙在左上角开放房间
- 两扇门都位于玩家的主要路径上

### 美术规范
- 玩家：用文字"我"表示，绿色背景
- 墙壁：用"█"表示
- 金钥匙：用"匙"表示，金色
- 银钥匙：用"匙"表示，银色
- 金门：用"门"表示，金色
- 银门：用"门"表示，银色
- 开启的门：用"门"表示，绿色
- 起点：用"出"表示
- 终点：用"终"表示

## 技术实现
- 纯HTML5 + CSS3 + JavaScript
- 无外部依赖
- 支持键盘和按钮控制

## 操作说明
- 方向键 / WASD：移动
- 空格键：切换场景
- 点击按钮：移动和切换场景
