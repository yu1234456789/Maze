# UE5 PvZ 2D Remake

基于扣子空间、使用ai开发的小型网页Html小游戏、主要用于验证玩法的可玩性
---

## 核心技术点

### 1. 场景切换
- **场景切换：** 保证场景的正常切换
- **地图差异：** 保证地图之间的差异化、能够让游戏正常进行。

### 2. 钥匙的生成
-**钥匙生成随机化**钥匙生成随机化、并保证钥匙能在玩家够的到的位置。

### 3. 钥匙的拾取和对应房门的打开
- **钥匙拾取：** 确保钥匙正常拾取并存在背包内。
- **对应房门的开启：**确保对应钥匙打开对应房门。
